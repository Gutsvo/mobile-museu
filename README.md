## Descrição do Projeto

Aplicativo Mobile desenvolvido em **Flutter** para fins educacionais, permitindo explorar a coleção pública do **Art Institute of Chicago** usando sua **API Open Access**.

O app possibilita:

- **Busca de obras de arte** por termo (ex.: título, artista)
- **Visualização de Destaques** na tela inicial
- **Sistema de Favoritos** com persistência de dados local
- **Exibição detalhada** de uma obra selecionada: imagem, título, artista e descrição
- **Navegação fluida** através de barra de navegação inferior

A API utilizada é **pública, gratuita e não requer autenticação**, permitindo integrações rápidas e eficientes.
